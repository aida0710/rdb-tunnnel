# RDS VPN Client

## Description
RDSを経由して通信を行うためのVPNクライアントです。

RDSを利用することで、全ての通信を詳細に記録し分析することが可能になります。

これにより、ネットワーク上のトラフィックを包括的に可視化し、潜在的な問題や異常を迅速に特定することができます。

また、詳細な通信ログを保持することで、インシデント発生時の調査や、コンプライアンス要件への対応も容易になります。
